# Analyze Metric Shift

## When to use it

Use this skill when the goal is to explain signal movement, not to build a full data model.

## Workflow

1. Identify the main shift, timeframe, and likely comparison baseline.
2. Note the strongest drivers that appear supported by the evidence.
3. Suggest the next cuts or validations that would reduce uncertainty.

## Output expectations

- Distinguish observation from interpretation.
- Keep the analysis brief and decision-oriented.
- End with targeted follow-up questions, not a generic research list.
